Roblox Zone

Roblox Zone adalah sebuah platform web interaktif yang dirancang khusus untuk komunitas pemain Roblox di Indonesia. Tujuan utama website ini adalah memecahkan masalah klasik yang sering dihadapi pemain: "Bingung mau main game apa hari ini?". Dengan antarmuka yang modern, penuh animasi, dan responsif, Roblox Zone menyediakan kurasi game terbaik, rekomendasi berdasarkan genre, hingga fitur Randomizer (Gacha) untuk memilih game secara acak dengan cara yang menyenangkan. Website ini memberikan pengalaman pengguna yang imersif melalui efek visual dan audio yang menarik.

2. Fitur Unggulan

Top Games Showcase: Menampilkan 3 game terpopuler saat ini dengan peringkat visual, memudahkan pengguna melihat tren terkini.

Genre Recommendation Engine:

Sistem navigasi berbasis genre (Fashion, Horror, Simulator, Adventure, Sport, Special Pick).

Tampilan overlay interaktif yang halus saat memilih genre.

Mekanisme card swapping (tukar kartu) untuk melihat detail game rekomendasi.

Game Randomizer (Gacha System):

Fitur unggulan berupa roda keberuntungan (roulette) untuk memilih game secara acak.

Dilengkapi efek suara ticking saat berputar dan efek winning saat berhenti.

Visual Reward: Efek hujan gambar (falling rain effect) dari game yang terpilih saat pengguna mendapatkan hasil.

Interactive About Us:

Profil anggota tim yang dikemas dalam kartu interaktif.

Modal Detail: Pop-up detail anggota tim dengan navigasi carousel (geser kanan/kiri) yang menampilkan nama, peran, jurusan, angkatan, dan tombol langsung ke profil Roblox mereka.

UI/UX Experience:

Custom Cursor: Kursor kustom dengan efek jejak (trail) yang dinamis.

Easter Egg: Fitur rahasia jika pengguna menekan tombol "O" pada keyboard (memunculkan suara dan efek visual "OOF").

Responsif: Layout yang menyesuaikan ukuran layar (desktop/tablet).

Audio: Integrasi efek suara pada interaksi tombol dan gacha.
